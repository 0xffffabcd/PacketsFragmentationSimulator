Fragmentation IP
================


Description
-----------

TP pour le cours de Réseau dont le but est de calculer (simuler?)
la fragmentation IP

Fonctionnalités:
----------------

* Support de plusieurs routeurs avec différents MTU
* Calcul du déplacement, longueur, et préservation du bit MF
